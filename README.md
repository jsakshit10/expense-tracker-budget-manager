#  Expense Tracker & Budget Manager (Python)

A simple command-line based Expense Tracker built using Python.  
This project helps users record daily expenses, categorize spending, and monitor their monthly budget.

It is designed to practice **Python OOP, encapsulation, and built-in data structures** like lists and dictionaries.

---

##  Features

- Add new expenses with amount, category, note, and date
- View all recorded expenses
- Calculate total spending
- View spending grouped by category
- Set a monthly budget
- Check remaining budget or budget overuse warning
- Menu-driven interactive console program

---

##  Concepts Used

This project demonstrates:

- **Classes & Objects**
- **Encapsulation**
- **Lists** (storing expense records)
- **Dictionaries** (category-wise spending summary)
- **Loops & Conditionals**
- **User Input Handling**
- **Basic Financial Logic**
